Datasets

These datasets were recorded around a building at the University of the Balearic Islands and can be used to assess the quality of visual loop closure or SLAM algorithms. See this paper for further details.

  • UIB Small Loop (53.82 MB)
  • UIB Large Loop (139.7 MB)
  • UIB Indoor Loop (42.06 MB)

Each file contains:

  • A directory called images. This directory includes an ordered sequence of images in PNG format. The size of the images is 300x240 and they were acquired at 1Hz.

  • A ground truth file in Matlab format, groundtruth.mat. This file includes a binary matrix where entry (i, j) is 1 if image i and image j can be considered as the same place, and 0 otherwise. Therefore, the i-th row represents the possible loop closures for image i. This ground truth was generated manually, and perhaps it should be modified if you want to use it as a reference.
