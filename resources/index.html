Datasets

Structural Defects

This dataset consists of a collection of images taken from structures and surfaces that present different kind of defects: corrosion, cracks, coating break-down, etc. The dataset also contains a collection of B/W images which resulted from the manual labelling of defects in the original images (black means defective area). This second collection is provided in case it is needed as ground truth.

  • Images (5.3 MB)
  • Ground truth (703.2 kB)
