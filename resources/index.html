Datasets

Structural Defects

This dataset consists of a collection of images taken from structures and surfaces that present different kinds of defects: corrosion, cracks, coating break-down, etc. The dataset also contains a collection of B/W images which resulted from the manual labelling of the defects in the original images (black means defective area).

  • Original image dataset (5.3 MB)
  • Ground truth for the original image dataset (703.2 kB)
  • Extended image dataset (11.2 MB)
  • Ground truth for the extended image dataset (1.5 MB)
